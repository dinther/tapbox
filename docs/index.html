tapbox — Web Installer

Flash tapbox firmware to your WT32-ETH01 directly from your browser. No software installation needed.

Requires Chrome or Edge (WebSerial support).
Windows users may need the CH340 driver if the board is not detected.

Prefer to flash manually? Download the binaries from Releases and run:
esptool.py --chip esp32 write_flash 0x1000 bootloader.bin 0x8000 partitions.bin 0x10000 firmware.bin
